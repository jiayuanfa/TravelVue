## 单页应用VS多页应用
### 多页面
- 每次后台都给返回一个html
- 页面切换比较慢
- 首页快
### 单页面
- 每次后台只返回路由
- 渲染由Vue来做，节省了很多http请求
- 首屏时间会稍微慢一点，SEO差
- 搜索引擎优化效果比较差
